# About
Example demonstrating how one might architect a single page application
Elm app.

# Dependencies MacOS
```bash
brew install node elm
npm install -g uglify-js@2.4.11
```

# Building
```
make serve
```

Now open `http://localhost:8000` in your browser.

# Viewing

Open index.html in your browser.
